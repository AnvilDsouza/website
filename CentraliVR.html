Centrali VR

Gaze or click on a dashboard to open. Say “Ali, show revenue trends” to query AI assistant.

Centrali VR: Key Features & Interactions
Feature Description User Interaction
Immersive 3D Room Users step into a spatial data environment with dashboards positioned around them. Move with WASD / gaze navigation.
Virtual Dashboards Each wall-mounted panel represents a BI report or metric board. Click or gaze to expand or open the dashboard in detail view.
Cross-Dashboard Filtering Real-time linking between dashboards for synchronized insights. Select a segment on one panel to update others instantly.
Ali — GPT Assistant AI avatar that answers questions or generates insights conversationally. Speak or type “Ali, show quarterly growth by region.”
Multi-Source Access Integrates dashboards, BI tools, and data lakes in one VR scene. Access all analytics via a single immersive portal.
